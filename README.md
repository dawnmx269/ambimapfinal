# 애매한 밤하늘 (Ambiguous Night Sky)

이 프로젝트는 사용자의 감정과 정체성을 별자리로 표현하는 인터랙티브 웹 애플리케이션입니다.

## 주요 기능

- 🌟 감정과 정체성을 나타내는 별들을 통한 자기 표현
- 🎨 동적으로 생성되는 개인화된 별자리
- 💫 감정 단어 조합을 통한 독특한 정체성 표현
- 📸 자신만의 별자리를 PNG로 저장 가능

## 기술 스택

- React
- Vite
- React Router
- HTML5 Canvas (html2canvas)

## 시작하기

1. 저장소 클론:
```bash
git clone [repository-url]
```

2. 의존성 설치:
```bash
npm install
```

3. 개발 서버 실행:
```bash
npm run dev
```

4. 빌드:
```bash
npm run build
```
